# Research Company Agent

AI-powered company research workflow that:
- Scrapes company websites using Firecrawl
- Analyzes content using Gemini AI
- Generates concise business summaries

## Features
- Website scraping
- AI company analysis
- Business overview generation
- Industry insights
- AI services detection

## Tech Stack
- Firecrawl
- Gemini 2.5 Flash
- Agent Builder
- Prompt Engineering

## Example Workflow
Input URL → Firecrawl → Gemini → Summary

## Example Companies
- OpenAI
- Lenovo
- Morningside AI

## Setup
1. Add Firecrawl API Key
2. Add Gemini API Key
3. Run workflow
